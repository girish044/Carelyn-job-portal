# 💼 Carelyn – Job Portal

Carelyn is a full-stack job portal web application built using Spring Boot and JavaScript.

## 🚀 Features

* User Signup & Login
* Apply for Jobs
* Profile Management
* Applied Jobs Tracking

## 🛠️ Tech Stack

* Java, Spring Boot
* MySQL
* HTML, CSS, JavaScript

## ▶️ Run Project

 Run Spring Boot
